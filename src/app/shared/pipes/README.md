# Shared Pipes (`src/app/shared/pipes`)

**Objetivo**
Mapear e formatar apresentações finais isoladas para dados recebidos em String (sanitização passiva), abstraindo do controller lógicas visuais puras.

**O que DEVE entrar aqui:**
- Formatadores de datas que fogem de escopo nativo, transformadores específicos de texto corporativo e sanitizadores de exibição puramente limpos, operando `pure: true`.
